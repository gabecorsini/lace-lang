Lace 🧵

A programming language for agents and the humans who supervise them.

Lace is a statically-typed scripting language where tool and fn are different things. Tools do I/O and are automatically logged, retryable, and auditable. Functions are pure. The runtime enforces the difference. 

@retry(max: 3)
tool fetch_user(id: Int) -> Result<String, String> {
    let response = Http.get("https://api.example.com/users/" ++ id.to_string())?
    let data = Json.parse(response)?
    Json.get(data, "name") |> Option.unwrap_or("unknown") |> Ok
}

fn greet(name: String) -> String {
    "Hello, " ++ name ++ "!"
}
🔒
Effect-safe

fn is pure. tool does I/O. The type checker enforces the difference and warns when you mix them.

📋
Auto-instrumented

Every tool call emits structured JSON logs automatically. No print-debugging. No manual tracing.

🥧
Pi-portable

Lean interpreter, bytecode VM, single-binary bundling. Runs on ARM. No GC pauses.
