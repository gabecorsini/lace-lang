Standard Library

Lace ships with a comprehensive standard library organized into namespaces. Core builtins are available globally; all other functions are accessed via their namespace prefix (e.g. String.len(s)).

Core

Core builtins are available in every Lace program without any import or namespace prefix. They cover output, assertions, and basic reflection.

FunctionSignatureDescription
print print(v: Any) -> Unit Prints a value to stdout without a trailing newline.
println println(v: Any) -> Unit Prints a value to stdout followed by a newline.
assert assert(cond: Bool) -> Unit Panics with an assertion error if cond is false.
assert_eq assert_eq(a: T, b: T) -> Unit Panics with a diff message if a != b.
to_string to_string(v: Any) -> String Returns the string representation of any value.
type_of type_of(v: Any) -> String Returns the runtime type name of a value as a string.
Example
let x = 42
println(x)                      // 42
println(to_string(x))           // "42"
println(type_of(x))             // "Int"
assert(x > 0)
assert_eq(x, 42)

String

Functions for inspecting, transforming, and parsing strings. Strings in Lace are UTF-8 encoded and immutable.

FunctionSignatureDescription
String.len len(s: String) -> Int Returns the number of characters in the string.
String.split split(s: String, sep: String) -> List<String> Splits the string on every occurrence of sep.
String.join join(list: List<String>, sep: String) -> String Joins a list of strings with sep between each element.
String.trim trim(s: String) -> String Removes leading and trailing whitespace.
String.starts_with starts_with(s: String, prefix: String) -> Bool Returns true if s begins with prefix.
String.ends_with ends_with(s: String, suffix: String) -> Bool Returns true if s ends with suffix.
String.replace replace(s: String, from: String, to: String) -> String Replaces all occurrences of from with to.
String.to_upper to_upper(s: String) -> String Returns an uppercase copy of the string.
String.to_lower to_lower(s: String) -> String Returns a lowercase copy of the string.
String.contains contains(s: String, sub: String) -> Bool Returns true if sub is a substring of s.
String.char_at char_at(s: String, i: Int) -> Option<String> Returns the character at index i, or None if out of bounds.
String.parse_int parse_int(s: String) -> Result<Int, String> Parses the string as an integer. Returns Err on failure.
String.parse_float parse_float(s: String) -> Result<Float, String> Parses the string as a float. Returns Err on failure.
Example
let s = "  Hello, World!  "
println(String.trim(s))                   // "Hello, World!"
println(String.to_upper("hello"))         // "HELLO"
println(String.split("a,b,c", ","))       // ["a", "b", "c"]
println(String.join(["x", "y", "z"], "-")) // "x-y-z"
println(String.contains(s, "World"))      // true
println(String.char_at("abc", 1))         // Some("b")

let n = String.parse_int("42")
match n {
  Ok(v)  => println(v)
  Err(e) => println("Parse error: " + e)
}

List

Ordered, immutable sequences. All transformations return a new list. Lists are generic — List<T> can hold any element type.

FunctionSignatureDescription
List.map map(list: List<A>, fn: A -> B) -> List<B> Applies fn to every element, returning a new list.
List.filter filter(list: List<T>, fn: T -> Bool) -> List<T> Returns elements for which fn returns true.
List.fold fold(list: List<A>, init: B, fn: (B, A) -> B) -> B Left-folds the list into a single accumulated value.
List.len len(list: List<T>) -> Int Returns the number of elements in the list.
List.push push(list: List<T>, v: T) -> List<T> Returns a new list with v appended at the end.
List.pop pop(list: List<T>) -> List<T> Returns a new list with the last element removed.
List.get get(list: List<T>, i: Int) -> Option<T> Returns Some(element) at index i, or None.
List.contains contains(list: List<T>, v: T) -> Bool Returns true if the list contains v.
List.sort sort(list: List<T>) -> List<T> Returns a sorted copy (ascending, natural order).
List.reverse reverse(list: List<T>) -> List<T> Returns the list in reversed order.
List.zip zip(a: List<A>, b: List<B>) -> List<(A, B)> Pairs elements by position. Truncates to the shorter list.
List.flatten flatten(list: List<List<T>>) -> List<T> Concatenates a list of lists into a single flat list.
List.sum sum(list: List<Float>) -> Float Returns the sum of all numeric elements.
List.max max(list: List<T>) -> T Returns the maximum element. Panics on empty list.
List.min min(list: List<T>) -> T Returns the minimum element. Panics on empty list.
List.is_empty is_empty(list: List<T>) -> Bool Returns true if the list has no elements.
List.join join(list: List, sep: String) -> String Joins a list of strings with a separator.
List.filter_map filter_map(list: List, f: Fn) -> List Maps each element through f, keeps only Some(x) results, and unwraps them.
Example
let nums = [1, 2, 3, 4, 5]

let doubled = List.map(nums, fn(x) => x * 2)     // [2, 4, 6, 8, 10]
let evens   = List.filter(nums, fn(x) => x % 2 == 0) // [2, 4]
let total   = List.fold(nums, 0, fn(acc, x) => acc + x) // 15

println(List.len(nums))       // 5
println(List.sum(nums))       // 15.0
println(List.reverse(nums))   // [5, 4, 3, 2, 1]

let pairs = List.zip([1, 2, 3], ["a", "b", "c"])  // [(1,"a"), (2,"b"), (3,"c")]

Map

Key-value store with string keys. Maps are immutable — all mutations return a new map. Useful for structured data and configuration.

FunctionSignatureDescription
Map.get get(map: Map, key: String) -> Option<T> Returns Some(value) for the key, or None.
Map.insert insert(map: Map, key: String, val: T) -> Map Returns a new map with key set to val.
Map.remove remove(map: Map, key: String) -> Map Returns a new map with key removed.
Map.keys keys(map: Map) -> List<String> Returns all keys as a list.
Map.values values(map: Map) -> List Returns all values as a list.
Map.contains_key contains_key(map: Map, key: String) -> Bool Returns true if the map contains the given key.
Map.len len(map: Map) -> Int Returns the number of key-value pairs in the map.
Example
let m = { "name": "Alice", "age": "30" }

println(Map.get(m, "name"))          // Some("Alice")
println(Map.contains_key(m, "city")) // false

let m2 = Map.insert(m, "city", "Paris")
println(Map.keys(m2))                // ["name", "age", "city"]
println(Map.len(m2))                 // 3

let m3 = Map.remove(m2, "age")
println(Map.keys(m3))                // ["name", "city"]

Option

Represents an optional value — either Some(v) containing a value, or None. Use Options instead of null to make absent-value handling explicit and safe.

FunctionSignatureDescription
Some Some(v: T) -> Option<T> Wraps a value in the Some variant.
None None -> Option<T> Represents the absence of a value.
Option.unwrap unwrap(opt: Option<T>) -> T Returns the inner value. Panics if None.
Option.unwrap_or unwrap_or(opt: Option<T>, default: T) -> T Returns the inner value, or default if None.
Option.is_some is_some(opt: Option<T>) -> Bool Returns true if the option contains a value.
Option.is_none is_none(opt: Option<T>) -> Bool Returns true if the option is None.
Option.map map(opt: Option<A>, fn: A -> B) -> Option<B> Applies fn if Some, passes None through.
Example
let a = Some(10)
let b = None

println(Option.is_some(a))            // true
println(Option.unwrap_or(b, 0))       // 0
println(Option.map(a, fn(x) => x * 2)) // Some(20)

match a {
  Some(v) => println("Got: " + to_string(v))
  None    => println("Nothing here")
}

Result

Represents the outcome of a fallible operation — either Ok(value) on success or Err(error) on failure. Functions that can fail (I/O, parsing, HTTP) return Results.

FunctionSignatureDescription
Ok Ok(v: T) -> Result<T, E> Wraps a success value.
Err Err(e: E) -> Result<T, E> Wraps an error value.
Result.unwrap unwrap(res: Result<T, E>) -> T Returns the success value. Panics on Err.
Result.unwrap_or unwrap_or(res: Result<T, E>, default: T) -> T Returns the success value, or default on Err.
Result.is_ok is_ok(res: Result<T, E>) -> Bool Returns true if the result is Ok.
Result.is_err is_err(res: Result<T, E>) -> Bool Returns true if the result is Err.
Result.map map(res: Result<A, E>, fn: A -> B) -> Result<B, E> Transforms the success value; passes errors through unchanged.
Example
let r = String.parse_int("123")

match r {
  Ok(n)  => println("Parsed: " + to_string(n))
  Err(e) => println("Error: " + e)
}

// Chaining with map
let doubled = Result.map(r, fn(n) => n * 2)
println(Result.unwrap_or(doubled, 0))   // 246

// Safe default
let bad = String.parse_int("abc")
println(Result.unwrap_or(bad, -1))      // -1

Math

Common mathematical functions and constants. All trigonometric and transcendental operations work with Float values.

FunctionSignatureDescription
Math.abs abs(n: Float) -> Float Returns the absolute value of n.
Math.floor floor(n: Float) -> Float Rounds down to the nearest integer value.
Math.ceil ceil(n: Float) -> Float Rounds up to the nearest integer value.
Math.sqrt sqrt(n: Float) -> Float Returns the square root of n.
Math.pow pow(base: Float, exp: Float) -> Float Raises base to the power of exp.
Math.log log(n: Float) -> Float Returns the natural logarithm (base e) of n.
Math.min min(a: Float, b: Float) -> Float Returns the smaller of two numbers.
Math.max max(a: Float, b: Float) -> Float Returns the larger of two numbers.
Math.round round(n: Float) -> Float Rounds to the nearest integer (half rounds up).
Math.PI PI: Float The mathematical constant π ≈ 3.14159265…
Math.E E: Float Euler's number e ≈ 2.71828182…
Example
println(Math.sqrt(16.0))          // 4.0
println(Math.pow(2.0, 10.0))      // 1024.0
println(Math.abs(-7.5))           // 7.5
println(Math.floor(3.9))          // 3.0
println(Math.ceil(3.1))           // 4.0
println(Math.round(3.5))          // 4.0
println(Math.PI)                  // 3.141592653589793

let circumference = 2.0 * Math.PI * 5.0
println(circumference)            // 31.41592653589793

Http

Simple synchronous HTTP client for making web requests. All functions return a Result<String, String> where the error string contains the status code or network error message.

FunctionSignatureDescription
Http.get get(url: String) -> Result<String, String> Sends an HTTP GET request and returns the response body.
Http.post post(url: String, body: String) -> Result<String, String> Sends an HTTP POST request with a plain-text body.
Http.post_json post_json(url: String, json: String) -> Result<String, String> Sends an HTTP POST with Content-Type: application/json.
Example
// Simple GET request
match Http.get("https://api.example.com/data") {
  Ok(body)  => println(body)
  Err(msg)  => println("Request failed: " + msg)
}

// POST with JSON payload
let payload = Json.stringify({ "name": "Alice", "score": 99 })
match Http.post_json("https://api.example.com/submit", payload) {
  Ok(resp)  => println("Saved: " + resp)
  Err(err)  => println("Error: " + err)
}

Json

Parse JSON text into Lace values and serialize Lace values back to JSON strings. Integrates naturally with the Http and File namespaces.

FunctionSignatureDescription
Json.parse parse(s: String) -> Result<Map, String> Parses a JSON string into a Lace Map. Returns Err on invalid JSON.
Json.stringify stringify(v: Any) -> String Serializes a Lace value to a compact JSON string.
Json.get get(map: Map, key: String) -> Option<Value> Retrieves a field from a parsed JSON map.
Json.keys keys(map: Map) -> List<String> Returns the top-level keys of a parsed JSON object.
Example
let raw = "{\"name\": \"Alice\", \"age\": 30}"

match Json.parse(raw) {
  Ok(obj) => {
    let name = Option.unwrap(Json.get(obj, "name"))
    println("Hello, " + to_string(name))    // Hello, Alice
    println(Json.keys(obj))                  // ["name", "age"]
  }
  Err(e)  => println("Parse error: " + e)
}

// Serialize back to JSON
let data = { "x": 1, "y": 2 }
println(Json.stringify(data))   // {"x":1,"y":2}

Env

Read and write process environment variables. Environment variables are global to the running process and affect child processes spawned after the call to Env.set.

FunctionSignatureDescription
Env.get get(key: String) -> Option<String> Returns the value of an environment variable, or None.
Env.set set(key: String, val: String) -> Unit Sets an environment variable for the current process.
Example
// Read an environment variable with a fallback
let home = Option.unwrap_or(Env.get("HOME"), "/tmp")
println("Home directory: " + home)

// Check for an API key
match Env.get("API_KEY") {
  Some(key) => println("Key loaded, length: " + to_string(String.len(key)))
  None      => println("Warning: API_KEY not set")
}

// Set a variable
Env.set("APP_ENV", "production")

File

Basic filesystem operations: reading, writing, and checking for existence. Paths may be absolute or relative to the working directory of the running process.

FunctionSignatureDescription
File.read read(path: String) -> Result<String, String> Reads the entire file as a UTF-8 string. Returns Err if the file cannot be read.
File.write write(path: String, content: String) -> Result<Unit, String> Writes (or overwrites) a file with the given content. Creates the file if it does not exist.
File.exists exists(path: String) -> Bool Returns true if a file or directory exists at the given path.
Example
// Check before reading
if File.exists("config.json") {
  match File.read("config.json") {
    Ok(contents) => {
      let cfg = Result.unwrap(Json.parse(contents))
      println("Config loaded")
    }
    Err(e) => println("Read error: " + e)
  }
} else {
  println("config.json not found, using defaults")
}

// Write a file
match File.write("output.txt", "Hello from Lace!\n") {
  Ok(_)  => println("Written successfully")
  Err(e) => println("Write failed: " + e)
}

Regex

Pattern matching and text extraction using regular expressions.

FunctionSignatureDescription
Regex.new new(pattern: String) -> Result<Regex, String> Compiles a regex pattern. Returns Err if the pattern is invalid.
Regex.is_match is_match(re: Regex, input: String) -> Bool Returns true if the input contains a match for the pattern.
Regex.find find(re: Regex, input: String) -> Option<String> Returns the first match as a string, or None.
Regex.find_all find_all(re: Regex, input: String) -> List<String> Returns all non-overlapping matches as a list of strings.
Regex.captures captures(re: Regex, input: String) -> Option<List<String>> Returns capture groups from the first match, or None if no match.
Regex.replace replace(re: Regex, input: String, replacement: String) -> String Replaces the first match with the replacement string.
Regex.replace_all replace_all(re: Regex, input: String, replacement: String) -> String Replaces all matches with the replacement string.
Regex.split split(re: Regex, input: String) -> List<String> Splits the input string on each match of the pattern.
Example
let re = Result.unwrap(Regex.new("\\d+"))

println(Regex.is_match(re, "price: 42"))   // true
println(Regex.find(re, "price: 42"))        // Some("42")
println(Regex.find_all(re, "a1 b22 c333")) // ["1", "22", "333"]

let email_re = Result.unwrap(Regex.new("(\\w+)@(\\w+\\.\\w+)"))
match Regex.captures(email_re, "user@example.com") {
  Some(groups) => println(groups)   // ["user", "example.com"]
  None         => println("no match")
}

let cleaned = Regex.replace_all(re, "foo123bar456", "NUM")
println(cleaned) // "fooNUMbarNUM"
